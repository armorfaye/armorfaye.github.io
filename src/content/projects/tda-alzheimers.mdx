## Problem & motivation

Early Alzheimer's diagnosis from MRI is hard because the relevant structure is
distributed and subtle. Topological data analysis (TDA) offers a different lens:
instead of looking only at intensities, it measures the *shape* of the data —
connected components, loops, and voids — which can persist as a robust signal.

## What I built

A diagnosis pipeline that extracts topological features from MRI-derived data
and feeds them into several models, with a controlled comparison across model
families on identical inputs.

## Technical approach

- **Vietoris–Rips filtrations** build a sequence of simplicial complexes from
  the data at increasing scales.
- **Persistence diagrams and persistence images** summarize which topological
  features appear and how long they persist, giving fixed-size, model-ready
  representations.
- **Model comparison.** Logistic regression, a random forest, and a CNN are
  trained and compared on the same features to isolate the contribution of the
  topological representation.
